# Indefinite Integral Studio 🎨

AI 기반의 동적 문제 생성 및 자가 검증 시스템을 갖춘 적분 학습 플랫폼

## 주요 기능
- **AI 하이브리드 모드**: AI를 이용한 실시간 문제 생성
- **온디맨드 상세 해설**: '해설 보기' 버튼 클릭 시 실시간으로 단계별 상세 풀이 생성
- **LaTeX 렌더링**: MathJax를 이용한 고퀄리티 수식 출력 (Pure Output Policy 적용)
- **적분상수(C) 엄격 채점**: 학습 목적에 따른 정밀한 채점 로직

## 🛠 기술 스택
- Vanilla JS, HTML5, CSS3
- Gemini AI API
- MathJax (LaTeX Rendering)
